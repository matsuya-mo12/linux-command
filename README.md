## 🍎 macでのLinuxコマンド

### 基本的なlinuxコマンド
| コマンド | 概要 | 主な使い方 |
| :--- | :--- | :--- |
| `pwd` | 現在の位置の確認 | `pwd` |
| `mkdir` | 新しいディレクトリ（フォルダ）を作成する | `mkdir ディレクトリ名` |
| `cd` | 作業するディレクトリを移動する | `cd ディレクトリ名  (cd ../は一つ前の階層に戻る)` |
| `ls` | 下層ファイル内のファイル確認 | `ls` |
| `ls -l` | 下層ファイル内のファイル詳細 | `ls -l` |
| `touch` | 中身が空の新しいファイルを作成する | `touch ファイル名.txt` |
| `cat` | ファイルの内容をターミナル画面に表示・確認する | `cat ファイル名.txt` |
| `rm` | ファイルの削除 | `rm ファイル名.txt` |
| `mv` | ファイル名の変更 | `mv 変更するファイル名 変更後のファイル名` |
| `mv` | ファイルの移動 | `mv 移動するファイル名 移動先ディレクトリ` |
| `mv` | ファイル移動し、ファイル名も変更 | `mv 移動するファイル名 ディレクトリ/変更後のファイル名` |
| `rm -r` | コマンドでファイルを下層も一緒に強制削除 | `rm -r ディレクトリ` |
| `rmdir` | 最下層のディレクトリの削除 | ` rmdir 最下層のディレクトリ` |

### 効率を上げるために役立つコマンド
| コマンド | 概要 | 主な使い方 |
| :--- | :--- | :--- |
| `history` | 過去に入力したコマンドの確認 | `history` |
| 入力中にtab | ファイル名入力中にtabをおすと補完してくれる | `例）ex + tab でexampleと補完する` |
| `caffeinate` | スリープ状態にさせない、閉じるときはcontrol+c | `caffeinate` |

### vi(テキストファイル編集ソフト)
#### viにはコマンドモード、文字入力モードの２つのモードがある
#### コマンドモードはメモ帳でいうと検索や閉じる、保存するなどの操作
#### 文字入力モードはメモ帳でいうと文字を入力するとき
| コマンド | 概要 | 主な使い方 |
| :--- | :--- | :--- |
| `vi` | 文字入力モードに切り替え | `vi ファイル名` |


#### 文字入力モード中
| コマンド | 概要 | 主な使い方 |
| :--- | :--- | :--- |
| `i` | 文字の入力ができるようになる、INSERTと書かれる<img width="380" height="326" alt="スクリーンショット 2026-06-28 22 56 49" src="https://github.com/user-attachments/assets/601df2b5-297e-4ed3-bb27-98e1a0428f51" /> | `i` |
| `x` | カーソルで選択して1文字削除 | `x` |
| `dd` | カーソルで選択して1行削除 | `dd` |
| ESCを2,3回押して`:q` | 文字入力していない場合コマンドモードに切り替え | ESCを2,3回押して`:q` |
| ESCを2,3回押して`:q!` | 保存しないで強制終了 | ESCを2,3回押して`:q!` |
| ESCを2,3回押して`:wq` | 上書き保存をして終了 | ESCを2,3回押して`:q!` |

### 検索
#### <mark>“検索を制すものがlinuxを制す”</mark>
| コマンド | 概要 | 主な使い方 |
| :--- | :--- | :--- |
| `view` | ファイル内の閲覧 | `view ファイル名` |
| `cat` | ファイル内の閲覧(一気に確認したい時) | `cat ファイル名` |
| `more` | ファイル内の閲覧(一画面分ずつ分割して確認したい時) | `more ファイル名` |
